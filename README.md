# n8n-nodes-wavix-api

This is an n8n community node that lets you use the [Wavix](https://wavix.com) SMS API in your n8n workflows.

[n8n](https://n8n.io/) is a [fair-code licensed](https://docs.n8n.io/sustainable-use-license/) workflow automation platform.

## Installation

Follow the [installation guide](https://docs.n8n.io/integrations/community-nodes/installation/) in the n8n community nodes documentation.

## Operations

### SMS

- **Send SMS** — Send an SMS/MMS message with support for media attachments, smart encoding, callback URLs, validity period, tags, and idempotency.
- **Get SMS** — Retrieve an SMS message by its UUID.

## Credentials

You need a Wavix API key. Generate one from your [Wavix dashboard](https://wavix.com) and add it as an API Key credential in n8n.
